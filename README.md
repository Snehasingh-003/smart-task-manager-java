# smart-task-manager-java

A simple GUI-based task management application built using Java Swing.

## Features
- Add and delete tasks
- Mark tasks as completed
- Assign priority (High, Medium, Low)
- Save tasks using file handling

## Technologies Used
- Java
- Swing (GUI)
- File Handling
- OOP Concepts

## How to Run
1. Clone the repository
2. Open in IntelliJ or Eclipse
3. Run MainUI.java
